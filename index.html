    Proyecto de Estadística Descriptiva con Python

    Bienvenidos al proyecto de estadística descriptiva con Python. Este proyecto está diseñado para ayudar a los estudiantes a entender los conceptos fundamentales de la estadística descriptiva y cómo se pueden aplicar en Python para el modelado de Machine Learning.

    Contenido del Proyecto

    • Recolección de datos
    • Análisis exploratório de datos (EDA)
    • Reprocesamiento de datos
    • Descripción de los datos
    • Visualización de datos
    • Interpretación de los resultados
    • Preparación para el modelado ML

    Tecnologías Utilizadas

    • Python
    • Numpy
    • Matplotlib
    • Pandas
    • Seaborn
    • Sklearn
    • Timeit
