Proyecto de Estadística Descriptiva con Python

Estadística Descriptiva

Bienvenidos al proyecto de estadística descriptiva con Python. Este proyecto está diseñado para ayudar a los estudiantes a entender los conceptos basicos de la estadística descriptiva y cómo se pueden aplicar en Python para el modelado de Machine Learning.

La estadística descriptiva se ocupa de recolectar, organizar, resumir y presentar datos de manera informativa. Su objetivo principal es describir las características principales de un conjunto de datos de manera cuantitativa. Por ejemplo, si tienes los datos de las calificaciones de un grupo de estudiantes, la estadística descriptiva te permitirá calcular la media, la mediana, la moda, el rango, la varianza, la desviación estándar, etc. Estos son todos ejemplos de medidas descriptivas que te dan una idea de la distribución de las calificaciones.
